Create screenshots from websites via json jobfiles
==================================================

This script creates screenshots from websites via `PhantomJS`_


JSON jobfile
++++++++++++

	{
	 "Output":"/tmp/foobar.png",
	 "ViewPort":"1280x1024",
	 "Url":"http://www.fotokasten.de"
	}
